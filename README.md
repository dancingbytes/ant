# Ant - simple BBCode editor with javascript ui

## Support

Ruby:  2.0+

Rails: 3.2+

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ant'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ant
